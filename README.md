

# SocialPulse

### Friendship Tracking & Interaction Reminder Dashboard

SocialPulse is a responsive web application that helps users track, manage, and visualize their interactions with friends. It focuses on maintaining meaningful relationships by monitoring communication frequency and encouraging regular check-ins.

## 🚀 Features

* 👫 **Friend Management** – View and manage multiple friend profiles in a responsive grid layout
* ⏰ **Contact Tracking** – Monitor days since last interaction with status indicators (overdue, almost due, on-track)
* ⚡ **Quick Check-In** – Log interactions (Call, Text, Video) instantly with toast notifications
* 📜 **Timeline History** – Automatically logs all interactions with date, type, and friend name
* 🔍 **Filtering System** – Filter timeline entries by interaction type (Call, Text, Video)
* 📊 **Interaction Analytics** – Visualize communication patterns using charts (Recharts)
* 📱 **Fully Responsive** – Optimized for mobile, tablet, and desktop devices
