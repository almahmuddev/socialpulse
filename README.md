# SocialPulse

### Social Interaction Tracker Dashboard

SocialPulse is a responsive web application that helps users track, manage, and visualize their social interactions with friends. It provides an intuitive interface to monitor communication frequency and maintain meaningful relationships.

---

## 🚀 Features

* 👫 **Friend Management** – View and track multiple friend profiles
* 📊 **Interaction Analytics** – Visualize call, text, and video activity using charts
* ⚡ **Quick Check-In** – Log interactions (Call, Text, Video) instantly
* 📜 **Timeline History** – See all past interactions in a structured timeline
* 🔍 **Filtering System** – Filter timeline by interaction type
* 📱 **Fully Responsive** – Works on mobile, tablet, and desktop

---

## 🛠️ Technologies Used

* **Next.js (App Router)**
* **React.js**
* **Tailwind CSS**
* **Recharts**
* **React Icons**
* **React Hot Toast**
* **LocalStorage (for timeline data)**
